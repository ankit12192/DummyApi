Add a user
      
POST /adduser

      
Payload

      {
      "Age": "33", 
      "Name ": "Alex", 
      "city": "Goa", 
      "email": "ankit.tiwari@test.com", 
      "id": "12"
       }
      
Response


{
    "data": {
        "Age": "33",
        "Email": "alex@alex.com",
        "city": "Goa",
        "created at": "Sun Oct  1 21:21:49 2017",
        "id": "12",
        "name": "Alex"
        }
}

delete a particular user 
      
DELETE /listusers/2


Response 

204


Update a user 
      
PUT /updateuser/2

      
Payload

      {
      "Age": "31", 
      "Name ": "amit", 
      "city": "Mumbai", 
      "email": "test@test.com", 
      "id": "12"
       }

{
    "data": {

        "email": "test@test.com",
        "id": "12",

    }
}
