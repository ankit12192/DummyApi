What is Dummyapi ?

DummyAPI brings to you an ever-growing set of API that come with a fully functional JSON response which can be used by any front end developer without hassles. You can also use DummyAPI to learn API automation or as an assignment for the interviews you are conducting at your organization. DummyAPI is designed for coders and will remain so forever.Dummyapi has all popular HTTP methods like GET, POST, PUT and DELETE
